# API Integration Dashboard

A lightweight Flask dashboard that aggregates data from REST APIs and displays key metrics in a clean web interface.

---

## 🚀 Features

- Fetches data from Salesforce, Stripe, and Google Analytics
- Displays metrics in a clean dashboard
- Includes user authentication
- Scheduled data refresh every hour
- Built with Flask and Jinja2

---

## 🛠️ Tech Stack

- Python 3.9+
- Flask
- Requests
- Jinja2
- Schedule

---
