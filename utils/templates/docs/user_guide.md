# User Guide – API Integration Dashboard

## Overview
A Flask dashboard that displays real-time metrics from Salesforce, Stripe, and Google Analytics.

## Setup
1. Clone the repo
2. Install dependencies
3. Add your API keys to config.json
4. Run: `python app.py`
5. Open: `http://localhost:5000`

## Customization
You can add more API integrations in `app.py`.
